## 4.5. `cook` step — single-output form

## 4.6. `cook` step — multi-output form

Defines `cook "a" "b" using { ... }` and `cook "a" "b" using >{ ... }`. Multi-output form requires a block-form `using` clause; `cook "a" "b" using "cmd"` is a parse error.
